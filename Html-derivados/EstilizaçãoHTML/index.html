Fundamentos de CSS

Aprenda a transformar páginas HTML simples em interfaces profissionais.

Por que aprender CSS?

CSS permite controlar cores, fontes, espaçamentos e layout das páginas web.
Com ele, um desenvolvedor consegue transformar um documento HTML simples
em uma interface visual organizada e agradável.

O que você aprenderá

  • Como aplicar estilos em páginas HTML
  • Propriedades tipográficas e fontes
  • Espaçamentos e alinhamentos
  • Uso de classes e IDs
  • Organização visual de conteúdos
