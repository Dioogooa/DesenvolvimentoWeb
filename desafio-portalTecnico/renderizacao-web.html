Construção do DOM e CSSOM

O que é o DOM?

DOM (Document Object Model) - É a representação em árvore do seu HTML. O navegador lê o HTML linha por linha e vai criando "nós" (tags, atributos, texto). Cada tag vira um objeto que pode ser acessado via JavaScript depois.

Enquanto o navegador lê o HTML, ele vai criando nós para cada elemento, criando uma estrutura de pais e filhos.

O que é CSSOM?

CSSOM (CSS Object Model) - É a mesma coisa, mas com os estilos! O navegador lê todas as regras CSS (nosso style, arquivos externos, etc) e monta uma árvore com os seletores e suas propriedades.

O navegador faz o mesmo processo, mas com as regras CSS.

Comparativo entre DOM e CSSOM

DOM CSSOM
Representa a estrturura HTML Representa as reglas de estilo CSS
É construído a partir de HTML É construído a partir de CSS
Cada tag vira um "nó" no DOM Cada seletor vira uma regra de no CSSOM
Pode ser manipulado com JS (JavaScript) Também pode ser acessado via JS. Mas é menos comum

Render Tree e Layout

Render Tree:

É a COMBINAÇÃO dos dois! O navegador junta o DOM (estrutura) com o CSSOM (estilo) e cria uma nova árvore contendo APENAS os elementos que serão realmente renderizados na tela. Elementos escondidos com display: none não entram aqui!

Layout (ou Reflow):

É o cálculo das posições! O navegador calcula: "Esse texto vai ocupar quanto de largura? Esse botão vai estar a quantos pixels do topo?" É como se ele desse coordenadas pra cada elemento.

Paint: A etapa final

O que é Paint?

Agora é desenhar de verdade! O navegador pega tudo que calculou no Layout e preenche os pixels na tela - cores, imagens, sombras, bordas... É como se fosse um pintor seguindo o desenho técnico.

As 5 Etapas do processo de renderização

  1. Construção do DOM
  2. Construção do CSSOM
  3. Combinação na Render TREE
  4. Cálculo do Layout
  5. Paint (pintura)

Exemplo pratico de codigo HTML:

                
                <!DOCTYPE html>
                <html>
                    <head>
                        <title>Minha página</title>
                    </head>
                    <body>
                        <h1>Título</h1>
                        <p>Meu primeiro texto</p>
                    </body>
                </html>
